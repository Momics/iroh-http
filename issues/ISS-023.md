# [ISS-023] `iroh-http-core` missing crates.io metadata fields

## Summary

The `iroh-http-core` `Cargo.toml` is missing `repository`, `documentation`, `keywords`, and `categories` fields required for a complete crates.io publish, which is a roadmap blocker.

## Evidence

- `crates/iroh-http-core/Cargo.toml:1` — missing required metadata fields
- `docs/roadmap.md:21` — roadmap requires these fields before v1.0

## Impact

The crate cannot be published to crates.io with a complete and discoverable metadata entry.

## Remediation

1. Add `repository`, `documentation`, `keywords`, and `categories` to `Cargo.toml`.

## Acceptance criteria

1. Crate metadata matches the roadmap checklist items.
