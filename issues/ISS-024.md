# [ISS-024] `iroh-http-discovery` missing crates.io metadata fields

## Summary

Same as ISS-023 but for `iroh-http-discovery`: the `Cargo.toml` is missing `repository`, `documentation`, `keywords`, and `categories` fields required for a complete crates.io publish.

## Evidence

- `crates/iroh-http-discovery/Cargo.toml:1` — missing required metadata fields
- `docs/roadmap.md:21` — roadmap requires these fields before v1.0

## Impact

The discovery crate cannot be published with a complete metadata entry.

## Remediation

1. Add `repository`, `documentation`, `keywords`, and `categories` to `Cargo.toml`.

1. Crate metadata matches the roadmap checklist items.
