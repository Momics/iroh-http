# [ISS-017] Trailer docs contain stale protocol internals

## Summary

The trailer-headers feature documentation references retired ALPN variants and old custom framing functions that no longer exist in the codebase.

## Evidence

- `docs/features/trailer-headers.md:53` — references retired ALPN variants and old custom framing functions
- `docs/internals/wire-format.md:40` — current wire docs define v2/hyper-native trailer handling

## Impact

Maintainers and users are misled about current protocol internals, increasing the risk of incorrect implementations and wrong mental models.

## Remediation

1. Rewrite the trailer feature doc using current hyper-native behavior only.

## Acceptance criteria

1. No references to retired trailer or ALPN variants remain in feature docs.
